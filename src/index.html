SocialFace

This is a Facebook-inspired social network. I worked on stuff like authentication with cookies and sessions, Web Sockets, uploading images, tagging contacts, and notifications for this application. React and Styled-Components were used to design the user interface, and Redux and the Context API were used to manage the state.

You may search, add, and delete contacts in Social Face, as well as tag them in your publications. A contact receives a notification every time you tag them. In addition, you can add photographs in your post. But it was an online chat that I really wanted to include. When you login to the application, you can see and communicate with the contacts who are online.

Jobfindr

Jobfindr is a web scraper that assists me in searching for web development jobs that meet certain criteria. JWT is used for authentication, Nodemailer is used for email delivery, and PostgreSQL is used for the database. The frontend is built in React, with the Context API for state management and Styled-Components for the user interface.

I decided to design an application that would assist me in one of the many employment portals that exist because of the large number of them that exist. Jobfindr searches www.ticjob.es twice a day for positions that suit my requirements, adds them to a database, and then sends me an email once a day with the new offers.

Face Recognizer

Based on the Clarifai API, this program recognizes faces. Cookies and sessions are used to maintain authentication, and React and TailwindCss are used to manage the user interface.

Typescript is used to build the backend. Docker containers are used to run the whole application. The application's production deployment is currently pending.
